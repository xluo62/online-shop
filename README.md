This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

路 Demo: https://uzi-live.herokuapp.com/
路 The goal of this project is to design and implement an E-Shopping website, which has completed data
persistence, data storage, user authentication, and other website shopping functions.
路 Used React.js as a front-end framework, asynchronous Redux as state management tool and back-end
services provided by Firebase for user authentication and database storage.
路 Implemented user authentication, addition and deletion of shopping cart items, database asynchronous
requests, and payment functions.
